## 🚀 Project Overview

This is a **Netflix Clone Web Application** built using **React.js**.  
It showcases movies, web series, and blog content with a modern UI similar to Netflix.

---

## ✨ Features

- 🎥 Movie listing UI (Netflix style)
- 📱 Fully responsive design
- 🧭 Navigation (Home, About, Blog)
- 🎬 "Watch Now" buttons
- 📝 Blog section
- ⚡ Fast performance using React
